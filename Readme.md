# Chat Application

A real-time chat application built with **React + TypeScript** for the frontend and **Node.js + TypeScript** for the backend.
The application allows users to communicate instantly through a modern and responsive interface.

---

## 🚀 Tech Stack

### Frontend

* React
* TypeScript
* Vite / Create React App
* CSS / Tailwind / Styled Components (update if needed)

### Backend

* Node.js
* Express
* TypeScript

### Database

* MongoDB

### Other Tools

* Socket.io (for real-time communication)
* JWT Authentication
* REST APIs

## ⚙️ Installation

Clone the repository:

```
git clone https://github.com/ArashadAli/Chat-Application.git
cd chat-application
```

### Install frontend dependencies

```
cd client
npm install
```

### Install backend dependencies

```
cd server
npm install
```

---

## ▶️ Running the Project

### Start Backend

```
cd server
npm run dev
```

### Start Frontend

```
cd client
npm run dev
```

---

## 🔐 Environment Variables

Create a `.env` file inside the **server** folder.

And take the reference from .env.example file
